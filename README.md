# Ali Husain Rizvi — Portfolio

Personal portfolio built with **Next.js 14**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Run locally

Node.js 18+ required.

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build for production

```bash
npm run build
```

Static export output is written to `out/` (configured for GitHub Pages / Vercel).

## Deploy

- **Vercel:** import repo and deploy (zero config)
- **GitHub Pages:** run `npm run build` and publish the `out/` folder

## Assets

- Profile photo: `public/ali-husain-rizvi.jpg`
- Resume PDF: `public/Ali Husain Rizvi Resume.pdf`

## Photo optimization

```bash
npm run optimize-photo
```

Processes `WhatsApp Image 2026-06-30 at 14.58.54.jpeg` into optimized JPG/WebP in `public/`.
